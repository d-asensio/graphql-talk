## Getting Started

1. Download the boilerplate

   ```sh
   git clone git@github.com:d-asensio/graphql-talk.git
   ```

2. Install dependencies

   ```sh
   yarn install
   ```

4. Start the webpack server. The server will run at [`localhost:3000`](http://localhost:3000).

   ```sh
   yarn start
   ```
